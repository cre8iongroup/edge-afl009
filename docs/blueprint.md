# **App Name**: ALPFA Convention Portal

## Core Features:

- Workshop Submission Form: Allow partners to submit workshop proposals with pre-defined fields like ALPFA Pillar, Session Format, Intended Audience, Session Title, Session Description, Objectives, and CPE credit option.
- User Authentication and Roles: Implement authentication system with different user roles (client, internal, regular user) and associated permissions to ensure data security and access control.
- Submission Dashboard: Display user submissions as cards with status indicators (e.g., 'waiting for approval,' 'needs new files') to allow users to track progress.
- Client Submission Output: Clients have a dedicated dashboard to view outputs of all submissions, which will contain all the collated and aggregated submission information.
- Notification System: Implement notifications for clients and internal teams to track new submissions, status updates, and other important events.
- AI Feedback Tool Bot: A generative AI powered bot where users can submit questions, to be routed to a team-accessible queue for answers. The bot will incorporate the contents of the submission when preparing its response using web-search and other internal data, when appropriate, to best help answer questions users might have.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to convey professionalism and innovation.
- Background color: Light grey (#F5F5F5) for a clean and modern look.
- Accent color: Electric purple (#009FE3) to highlight key actions and interactive elements.
- Headline Font: 'Poppins' sans-serif font for headers and titles, ensuring a contemporary and readable design.
- Body Font: 'PT Sans' sans-serif font for body text, complementing Poppins for readability.
- Use consistent and professional icons to represent different submission statuses and categories.
- Design a clean and intuitive dashboard layout for easy navigation and data accessibility for users.
- Subtle transition animations when status updates happen in the submission dashboards.
